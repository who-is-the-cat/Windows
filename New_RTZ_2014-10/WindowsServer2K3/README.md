### Обновление часовых поясов для Windows Server 2003

#### Установке на остальные рабочие станции
* Windows2K3_RTZ-2.reg - ключ реестра, который удаляет ветку текущего (выставленного) часового пояса.
* Windows2K3_RTZ-4.reg - ключ реестра, который добавляет ветку с текущим (выставленным) часовым поясом.

* **Windows2K3_RTZ.cmd** - скрипт автоматического обновления часовых поясов.

#### Отмена автоматического перевода времени через реестр

* Windows2K3_RTZ_Dis.reg - ВЫключить автоматический переход на летнее время и обратно.
